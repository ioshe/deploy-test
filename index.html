PROJECTS
AI를 활용한 초개인화 카드 발급 서비스
2024.03~2024.05 - 팀 프로젝트

프로젝트의 주된 목적은 개인화된 신용카드 추천 서비스를 제공하는 것이었습니다.
이를 위해, AI 기술을 활용한 군집 기반 추천 시스템과 초개인화된 카드 디자인 생성 기능을 개발하였습니다.
특히, AWS 환경에서의 배포와 CI/CD 구축, 그리고 Spring Boot와 Spring Security를 사용한 보안 기능 구현에 집중했습니다.

주 역할

  1. AWS를 통한 서버 인프라 구축
  2. Jenkins, Docker 를 활용한 자동화 CI/CD 배포
  3. Git으로 프로젝트 기능별 분산형 버전관리 시스템 기획
  4. 사용자, 시스템 로그 수집(ELK, Prometheus, Grafana)
  5. 노션 기반 협업 워크스페이스 구축
  6. JWT를 이용한 인증 시스템 구현

frontend : bootstrap JSP

backend : Springboot SpringSecurity Mysql flask ELK

deploy : AWS Jenkins Docker

칭찬게시판
2024.02 - 팀 프로젝트

이 프로젝트는 사용자 간에 서로 칭찬을 공유하고 긍정적인 상호작용을 장려하기 위한 온라인 칭찬 게시판 시스템을 개발하는 것이 목표였습니다.
이 시스템은 Spring Boot를 기반으로 하여 구축되었으며, AWS 클라우드 서비스를 이용하여 배포되었습니다.
게시판은 사용자가 서로를 칭찬하는 글을 작성하고, 이를 통해 포인트를 얻을 수 있는 기능을 포함하고 있습니다.
사용자는 자신의 글이나 다른 사람의 글에 좋아요를 누르고, 이에 따라 포인트가 적립되는 시스템이 마련되어 있습니다.

주 역할

  1. JPA를 이용한 데이터베이스 설계 및 연동
  2. 세션 관리를 통한 포인트 로직 개발

frontend : bootstrap JSP

backend : Springboot Mysql

deploy : AWS Jenkins Docker
